# [78] 子集

给定一组不含重复元素的整数数组 nums，返回该数组所有可能的子集（幂集）。

说明：解集不能包含重复的子集。

示例:

输入: nums = [1,2,3]

输出:

[

[3],

[1],

[2],

[1,2,3],

[1,3],

[2,3],

[1,2],

[]

]

经典回溯问题，使用前序遍历，取出每一个第一次走到的路径，入结果数组即可。

var subsets = function(nums) {
  const res = [];
  backtrack(nums, [], 0);
  return res;

  function backtrack(nums, path, start) {
    // 多叉树的前序遍历
    res.push([...path]);
    for (let i = start; i < nums.length; i++) {
      path.push(nums[i]);
      backtrack(nums, path, i + 1);
      path.pop();
    }
  }
};
